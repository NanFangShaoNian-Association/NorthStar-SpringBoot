# 北极星宠基础后端环境
## 快速启动指南
### 环境配置
1. 更改application.yml中的环境为prod
2. 将application-prod.yml中的数据库账号密码配置进去
3. 配置Redis地址，默认本地无密码
3. 导入sql/pet_platform.sql文件
4. 启动项目
